Context Window

Most models have a 200k token context window
(some models have 1M tokens)

Once the context is <50% full, performance degrades

Make sure to keep the chat free of distractions like:

  • Back and forth conversations
  • Reading tons of files
  • Lots of tool calls
  • Inline plans

Context Curation

🧹 What can we do to keep the context clean? 🧹

  • Store plans in .md files
  • Start new chat sessions frequently
  • Revert to last message in case of bad results

Fast Context

Why it's great

  • It's a subagent → context stays clean
  • Extremely fast ⚡
